# Bus Company Management System

## Course
SWE201 - Object Oriented Programming

## Description
This project is a console-based application developed in C#.
It demonstrates object-oriented programming principles through a simple bus company management system.

## Classes
- Bus: Represents buses owned by the company.
- Driver: Represents drivers working for the company.
- Route: Represents routes operated by the company.
- BusCompany: Manages buses, drivers, and routes.

## Object-Oriented Design
- Encapsulation is used via properties.
- Association is implemented using collections.
- Constructors initialize class objects.

## Conclusion
This project was designed to model a real-world bus company using object-oriented principles.
